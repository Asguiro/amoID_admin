# Checklist

- identité AMO et tokens ;
- clarté du titre, périmètre et action ;
- hiérarchie et densité ;
- composants partagés ;
- tableaux et filtres ;
- loading/empty/error ;
- desktop/tablette/mobile ;
- clavier et focus ;
- contraste ;
- données sensibles ;
- absence de template SaaS générique.

# Sortie

Classer les problèmes en bloquant, important, amélioration. Proposer des corrections concrètes fichier par fichier.
